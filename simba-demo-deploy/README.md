# Simba Connect — AI Channel Demo

A single-file, mobile-optimised interactive demo of the Simba Connect WhatsApp channel, built by Berelo.

## Contents
- `index.html` — the complete demo (no build step, no dependencies).
- `vercel.json` — static hosting config for Vercel.

## Deploy
This is a static site. On Vercel, no framework preset is needed — it serves `index.html` directly. Any push to the `main` branch redeploys automatically once the repo is connected to a Vercel project.

## Share
Once live, share the URL (not the file) in WhatsApp, and turn it into a QR code for the room.
